# Gestor de Gastos Mensuales en Python

Programa en consola que permite registrar y administrar gastos mensuales.

## Funciones

- Registrar gastos
- Mostrar gastos guardados
- Calcular total de gastos

## Cómo usarlo

1. Ejecutar el archivo gestor_gastos.py
2. Elegir una opción del menú
3. Seguir las instrucciones en pantalla
